## Что такое Sendsay Android SDK?

Sendsay Android SDK позволяет интегрировать ваше мобильное приложение с CDP Sendsay и отслеживать поведение пользователей. С его помощью вы можете отправлять push-уведомления, фиксировать события и свойства клиентов.

SDK [доступен на GitHub](https://github.com/sendsay-ru/sendsay-mobile-sdk-android) с открытым исходным кодом и включает [пример приложения](docs/example-app.md), который можно использовать для тестирования и изучения возможностей SDK.
